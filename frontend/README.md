# 셋업 가이드

### 패키지 설치 : pnpm i

### storybook 로컬실행 : pnpm run storybook

### storybook 빌드 : pnpm run build-storybook

### 로컬 실행 : pnpm run start

### 로컬 빌드 : pnpm run build
